# Simple video and audio chat app 

## Technologies used 
* ReactJs for frontend
* NodeJS and SocketIO for signalling
* WebRTC API


### For local development, clone this repository and open terminal in home directory of the cloned project

#### For frontend part
```cd frontend```<br>
```npm i```<br>
```npm run dev```

#### For backend part 
```cd backend```<br>
```npm i```<br>
```npm run start```
